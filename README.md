## Billing AR Data Analysis

NERC-compliant revenue analysis and customer risk monitoring for Electricity Distribution Company

## 📊 Project Overview

# Features

Month-on-Month NERC Compliance tracking
Feeder Performance Scorecards with Risk Rating
Automated PDF reports for NERC submission
Customer lookup with demand notice generation
Revenue Protection Alerts - meter bypass, zero-pay detection
Major Feeder monitoring - Kumbotso 33KV, Dan Agundi, etc
