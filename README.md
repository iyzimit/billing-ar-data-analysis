# Billing AR Data Analysis

Analysis of Accounts Receivable data to track aging, DSO, collections efficiency, and customer payment behavior.

## 📊 Project Overview

This repo contains Python notebooks and scripts for analyzing billing/AR data. Key outputs:
- **AR Aging Reports**: 0-30, 31-60, 61-90, 90+ day buckets
- **DSO Calculation**: Days Sales Outstanding trends
- **Collections KPIs**: % collected, overdue amount, top debtors
- **Customer Segmentation**: Payment behavior patterns
